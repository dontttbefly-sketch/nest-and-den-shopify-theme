# PUPKIT 小型犬玩具实验室

PUPKIT 是一个作品集用途的电商独立站概念项目，主题是“小型犬玩具”。项目不依赖真实商品图片，而是用 CSS 图形、动效和交互来完成品牌表达、商品浏览、筛选和详情体验。

核心口号：

> 把玩心放大，把风险做小。

## 项目定位

这个站点面向作品集展示，重点不是模拟完整交易系统，而是展示一个细分电商品牌如何通过视觉语言、页面节奏和交互细节建立可信度。

- 品牌：PUPKIT 小型犬玩具实验室
- 品类：小型犬玩具、小型犬互动玩具、慢食益智玩具、低噪陪伴玩具
- 关键词：用心、安全、品质、轻量、可清洗、咬感分级
- 展示目的：电商独立站视觉设计、交互设计、前端静态实现
- 商品状态：品牌与商品均为虚构，不用于真实销售

## 页面结构

### 首页 `index.html`

首页负责建立品牌第一印象，并把用户自然带进商品场景。

- Hero：品牌入口、CSS 玩具舞台、轻量鼠标视差。
- 产品理念：展示“用心 / 安全 / 品质”的品牌标准核心。
- 按玩法逛：按耐咬、益智藏食、抛接追逐、低噪陪伴切换商品。
- 材质语言：用简短判断表达商品选择逻辑。

### 商店页 `shop.html`

商店页是主要商品浏览界面，强调筛选效率和详情质感。

- 16 件虚构商品。
- 按玩法气质、身体尺度、购买心情组合筛选。
- 舒展 / 紧凑两种货架密度。
- 点击商品卡打开右侧大详情抽屉。
- 加入玩具袋时有商品图形飞入购物袋的仪式动画。

### 玩法指南 `playbook.html`

玩法指南展示每件商品的玩法框架，让商品不只是“被购买”，也有清晰的使用场景。

- 左侧商品目录。
- 右侧视频框架占位。
- 商品标题、标签、短句和步骤随目录切换。
- 移动端自动改成纵向浏览。

## 设计语言

PUPKIT 的视觉方向是“温暖实验室”：奶油纸底、黑色线条、柠檬黄重点色，加上红、蓝、绿作为产品信号点。

- 背景：纸面网格、柔和暖黄 / 浅蓝 / 淡红渐变。
- 图形：用 CSS 绘制玩具轮廓，避免依赖真实商品摄影。
- 卡片：低圆角、清晰边界、轻阴影，保持作品集里的高级感。
- 动效：鼠标视差、卡片 hover、高亮切换、筛选淡入淡出、购物袋飞行动画。
- 文案：短句表达理念，不写说明书式提示。

## 交互功能

- 首页 Hero 图形跟随鼠标轻微移动。
- 产品理念卡片 hover / focus / click 时同步更新中心核心文案和颜色。
- 首页“按玩法逛”可筛选商品，并带平滑渐变出场。
- 商店页组合筛选真正匹配商品属性。
- 商品卡可点击打开右侧详情浮窗。
- 加入玩具袋后，商品图形从页面中飞入购物袋。
- 玩法指南目录可切换不同商品的玩法内容。
- 移动端无横向滚动，导航和购物袋按钮保持可用。

## 技术实现

项目是纯静态站，方便直接预览和部署。

```text
.
├── index.html
├── shop.html
├── playbook.html
├── assets
│   ├── app.js
│   ├── favicon.svg
│   └── styles.css
└── README.md
```

- HTML：三页静态结构。
- CSS：完整视觉系统、响应式布局、CSS 玩具图形与动效。
- JavaScript：商品数据、筛选逻辑、购物袋、详情抽屉、玩法指南切换、鼠标互动。
- 无框架、无构建工具、无真实图片依赖。

## 本地预览

直接打开 `index.html` 可以查看页面。为了让路径行为更接近真实部署，也可以启动本地静态服务器：

```bash
python3 -m http.server 8767 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:8767/index.html
```
